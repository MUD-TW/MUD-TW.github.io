## 分支狀況

本專案共有三個分支，分別負責不同用途：

- `main` 存放專案說明
- `docs` 存放站點原始碼
- `gh-pages` 託管由 GitHub Actions 根據 ‵docs` 分支存放的內容，自動構建的靜態頁面文件
